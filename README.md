# MealSpy

MealSpy is a free web app that lets you compare food cart prices from **Swiggy** and **Zomato** by pasting a shared cart link. The app detects the platform and prepares for future price comparison features.

> ⚠️ Not affiliated with Swiggy or Zomato.

## Features (MVP)
- Pastes shared cart link
- Detects whether it's from Swiggy or Zomato
- Shows detected platform

## Tech Stack
- React
- Vite
- Hosted on Vercel (Free)
